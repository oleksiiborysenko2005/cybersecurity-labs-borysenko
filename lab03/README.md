# КОМПЛЕКСНА СИСТЕМА ЗАХИСТУ (AES + Стеганографія)

Цей проект демонструє двоетапний комплексний захист даних: поєднання криптографії та стеганографії для маскування файлів.

---

## Використані Технології

* Криптографія: AES-128 (модуль Fernet).
* Маскування: Метод LSB (Least Significant Bit) за допомогою бібліотеки Pillow (PIL).
* Аналітика: Збір метрик часу виконання та зміни розміру файлів.

---

## Встановлення та Запуск

### 1. Вимоги

Для роботи потрібні наступні бібліотеки:

pip install cryptography pillow

### 2. Запуск

1.  Розмістіть файл lab_complex.py, test.txt та photo.png в одній папці.
2.  Запустіть скрипт:

python lab_complex.py

---

## Функціонал

Програма пропонує два основні режими роботи.

### 1. Захистити файл

Виконує послідовний захист:

1.  Шифрування: Текстовий файл шифрується за допомогою AES. Ключ генерується на основі Логіна та Пароля.
2.  Стеганографія: Зашифрований токен ховається в зображенні-носії.
3.  Вивід: Генерація файлу stego_photo.png та Аналітичний звіт.

### 2. Відновити файл

Виконує зворотний процес:

1.  Вилучення: Зі stego_photo.png витягується зашифрований токен.
2.  Розшифрування: Токен розшифровується за допомогою Логіна та Пароля.
