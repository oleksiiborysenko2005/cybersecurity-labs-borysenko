Комплексний Захист Файлів (AES + LSB Стеганографія)
Цей проект реалізує вимоги лабораторної роботи з дисципліни "Захист інформації" та демонструє двоетапний комплексний захист даних, поєднуючи криптографію та стеганографію.

Програма дозволяє зашифрувати вміст файлу і приховати його в зображенні-носії, щоб замаскувати факт передачі секретних даних.

Використані Технології
Мова: Python 3.x

Криптографія: cryptography.fernet (стандарт AES-128)

Стеганографія: Pillow (PIL) (метод LSB - Least Significant Bit)

Аналітика: Збір метрик часу виконання та зміни розміру файлів.

Встановлення та Запуск
1. Вимоги
Для роботи потрібні наступні бібліотеки:

Bash

pip install cryptography pillow
2. Запуск
Розмістіть файл lab_complex.py та файли-зразки (test.txt, photo.png) в одній папці.

Запустіть скрипт:

Bash

python lab_complex.py
Функціонал
Програма пропонує два основні режими роботи:

1. Захистити файл (Шифрування → Стеганографія)
Виконує послідовний захист:

Криптографія (Етап 1): Текстовий файл шифрується за допомогою AES, де ключ генерується на основі Логіна та Пароля користувача.

Стеганографія (Етап 2): Зашифрований токен приховується в зображенні-носії (photo.png) за допомогою методу LSB.

Вивід: Генерація файлу stego_photo.png та Аналітичний звіт з метриками (час шифрування, час приховування, зміна розміру).

2. Відновити файл (Вилучення → Розшифрування)
Виконує зворотний процес:

Вилучення: Зі stego_photo.png витягується зашифрований токен.

Розшифрування: Токен розшифровується за допомогою повторно введених Логіна та Пароля.

Комплексний Захист (Висновок)
Ця система реалізує подвійний захист:

Маскування: Факт передачі секрету прихований у візуально невинному файлі.

Конфіденційність: Навіть якщо прихований токен буде виявлений, він залишиться нечитабельним без правильного ключа.
